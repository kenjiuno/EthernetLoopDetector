EthernetLoopDetector
====================

EthernetLoopDetectorは、WinPcapを使用し、ネットワークに存在し得るループの検出を支援致します。

必要ソフト
----------

- Microsoft .NET Framework 4.0
- WinPcap 4.1.3以降
